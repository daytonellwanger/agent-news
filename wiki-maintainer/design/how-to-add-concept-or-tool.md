# How to Add a Concept or Tool Page

Use this when a concept or tool needs its own wiki page and doesn't have one yet.

## Process

1. Search the web for information about the concept or tool if the existing wiki doesn't already cover it sufficiently.

2. Create the page at `concepts/<slug>.md` or `tools/<slug>.md` following the format in `design/wiki-structure.md`. Use the "What Belongs in Concepts vs. Tools" section to decide which directory.

3. Update `index.md` with the new entry.

4. Link from the new page to related existing pages, and update those existing pages to link back where relevant.
